# ⚡ 快速启动 - "使用默认数据"按钮

## 🎯 最快的方法（3步）

### 方式 A：使用 UI 按钮
**位置**：简历生成器左侧导航栏的最底部

```
访问: http://localhost:5173/resume-builder
↓
左侧导航栏 → 向下滚动 → 底部蓝色按钮【使用默认数据】
↓
点击 → 确认 → 完成
```

### 方式 B：使用快捷键（推荐！）⭐
```
在任何页面按下: Ctrl + Shift + D
↓
弹出确认对话框
↓
确认 → 完成
```

**优势**：
- ✓ 无需找按钮
- ✓ 速度快
- ✓ 在任何页面都有效
- ✓ 在 `/resume-builder` 页面最有用

## 🔍 看不到按钮？

**不用担心！使用快捷键:**
```
Ctrl + Shift + D
```

### 快速检查清单
- [ ] 确保在 `/resume-builder` 页面
- [ ] 左侧导航栏是否完全显示？
- [ ] 向下滚动左侧面板查看底部
- [ ] 或使用快捷键 **Ctrl + Shift + D**
- [ ] 刷新页面 (Ctrl + F5)

## 💾 使用后会发生什么？

✓ 简历被填充示例数据：
  - 姓名、邮箱、电话等个人信息
  - 2条工作经历
  - 1条教育背景
  - 8个技能
  - 1个项目等

✓ 数据自动保存到浏览器数据库

✓ 预览窗口立即更新

## ⚠️ 重要注意事项

- **现有数据会被覆盖** - 操作前系统会提示警告
- **可以后退** - 按浏览器后退按钮不会恢复，但可以手动删除不需要的数据
- **本地保存** - 数据仅保存在您的浏览器中，不上传到服务器

## 🆘 遇到问题？

| 问题 | 解决方案 |
|------|--------|
| 按钮或快捷键都不工作 | 刷新页面 + Ctrl+F5 清除缓存 |
| 快捷键显示错误 | 检查是否启用了其他应用的快捷键冲突 |
| 点击无反应 | 等待页面完全加载，检查 F12 控制台 |
| 数据未更新 | 关闭/重新打开页面 |

## 🎮 快捷键列表

|-------|------|
| **Ctrl + Shift + D** | 填充默认数据 |
| **Ctrl + F5** | 清除缓存后刷新 |
| **F12** | 打开开发者工具 |
