## ✨ 使用步骤

### 方案A：使用左侧导航栏按钮

1. 打开简历生成器
2. **向下滚动左侧导航栏**到底部
3. 看到蓝色**"使用默认数据"**按钮
4. **点击按钮**
5. 在确认对话框中点击**"确认填充"**
6. ✅ 简历自动填充示例数据

### 方案B：使用编辑区按钮

1. 打开简历生成器
2. 在中间编辑区查看顶部
3. 看到黄色**"填充示例"**按钮（在"AI助手"左边）
4. **点击按钮**
5. 在确认对话框中点击**"确认"**
6. ✅ 简历自动填充示例数据

## 🔍 如果还是看不到按钮

### 检查清单：

- [ ] 确保已经打开"简历生成器"页面 (`/resume-builder`)
- [ ] 尝试**刷新浏览器** (F5 或 Ctrl+R)
- [ ] 清除浏览器缓存 (Ctrl+Shift+Delete)
- [ ] 检查浏览器控制台是否有错误 (F12)
- [ ] 尝试**最小化浏览器窗口**，看左侧导航栏是否显示按钮

## 📝 填充数据内容说明

点击后会填充以下内容：

| 模块 | 数据量 |
|------|--------|
| 个人信息 | ✓ 姓名、邮箱、电话等 |
| 个人简介 | ✓ 一段完整的自我介绍 |
| 工作经历 | ✓ 2条示例工作经历 |
| 教育背景 | ✓ 1条硕士学历 |
| 技能特长 | ✓ 8个技能 |
| 项目经历 | ✓ 1个项目示例 |
| 证书资质 | ✓ 2个认证 |
| 语言能力 | ✓ 3种语言 |

## ⚠️ 重要提示

**填充默认数据会覆盖现有数据！**

- 操作前会显示确认对话框
- 建议在填充前保存重要数据
- 数据会自动保存到本地数据库
- 可通过浏览器撤销功能恢复（Ctrl+Z）

## 🐛 故障排查

### 问题1：点击按钮后没有反应

**解决方案：**
- 检查浏览器控制台（F12）查看是否有错误
- 确保简历 Store 已正确初始化
- 刷新页面后重试

### 问题2：按钮显示但无法点击

**解决方案：**
- 检查是否有其他模态对话框覆盖
- 尝试清除浏览器缓存
- 检查浏览器是否允许脚本执行

### 问题3：填充后数据没有显示

**解决方案：**
- 刷新页面查看数据是否已保存
- 检查浏览器 IndexedDB 是否已启用
- 查看浏览器控制台错误日志
