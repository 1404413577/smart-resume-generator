# 浏览器原生打印功能说明

## 功能概述

现在简历导出已改为使用**浏览器原生打印功能**，这样可以：
- ✅ 保持最佳的字体渲染效果
- ✅ 保留原始的文字间距和排版
- ✅ 避免 html2canvas 导致的字体挤压问题
- ✅ 生成更高质量的PDF文件
- ✅ 支持所有浏览器的打印预览和设置

## 使用方法

### 方法1：点击"导出PDF"按钮
1. 在简历编辑器中点击右上角的**"导出PDF"**按钮
2. 浏览器会打开打印对话框
3. 在"目标打印机"或"打印机"下拉菜单中选择**"另存为PDF"**或**"Save as PDF"**
4. 选择保存位置并命名文件
5. 点击**"保存"**按钮完成PDF导出

### 方法2：使用键盘快捷键
- **Windows/Linux**: 按 `Ctrl + P`
- **Mac**: 按 `Cmd + P`
- 然后按照方法1的步骤3-5操作

## 打印设置建议

在打印对话框中，建议进行以下设置以获得最佳效果：

### 基础设置
- **页面尺寸**: A4
- **方向**: 纵向（Portrait）
- **边距**: 无（None）或最小
- **缩放**: 100%（默认）

### 高级设置（可选）
- **背景图形**: 勾选（确保背景色正确显示）
- **页眉和页脚**: 不勾选（避免显示浏览器默认的页眉页脚）

### Chrome浏览器专用设置
1. 点击"更多设置"
2. 勾选"背景图形"
3. 取消勾选"页眉和页脚"
4. 边距设置为"无"

## 常见问题

### Q: 为什么要改用浏览器原生打印？
A: html2canvas 会将页面转换为图片，可能导致字体间距异常。浏览器原生打印直接渲染HTML，保持最佳效果。

### Q: 打印的排版和预览不一样怎么办？
A: 这是最常见的问题！请按以下步骤排查：

1. **检查浏览器缩放**：确保浏览器缩放为100%（按Ctrl+0重置）
2. **检查打印缩放**：在打印对话框中，确保"缩放"设置为100%
3. **检查页边距**：将边距设置为"无"或"最小"
4. **清除缓存**：按Ctrl+Shift+Delete清除缓存后重试
5. **使用推荐浏览器**：Chrome或Edge效果最佳

**核心原则**：现在的打印功能完全不修改预览样式，打印效果应该与预览完全一致。

### Q: 如何确保背景色正确显示？
A: 在打印对话框中勾选"背景图形"选项（Chrome/Edge）。

### Q: 可以预览打印效果吗？
A: 可以！打印对话框左侧会显示实时预览。

### Q: 如何调整页边距？
A: 在打印对话框的"更多设置"中选择边距选项（无/最小/默认/自定义）。

### Q: 打印出来的页面太小/太大怎么办？
A: 在打印对话框中调整"缩放"选项，建议保持100%。

### Q: 为什么按Ctrl+P会自动打开打印对话框？
A: 这是为了方便用户快速导出PDF。如需禁用，请告知我们。

### Q: 如何调试打印问题？（开发者）
A: 在开发模式下，打开控制台输入：
```javascript
// 查看打印环境信息
window.printDebug.report()

// 检查样式冲突
window.printDebug.checkConflicts()

// 比较元素样式
window.printDebug.compare('resume-preview')

// 预览打印模式
window.printDebug.preview()
```

## 技术实现

### 核心文件
- `src/utils/pdf/browserPrint.js` - 浏览器打印核心逻辑
- `src/styles/print.css` - 打印专用CSS样式
- `src/components/layout/AppHeader.vue` - 顶部导出按钮
- `src/views/ResumeBuilderView.vue` - 编辑器导出功能
- `src/App.vue` - 全局快捷键支持

### 关键特性
1. **智能样式注入**: 自动添加打印专用CSS
2. **元素隔离**: 只打印简历内容，隐藏UI控件
3. **多页支持**: 自动处理多页简历的分页
4. **事件监听**: 打印完成后自动清理临时元素
5. **快捷键支持**: Ctrl+P / Cmd+P 快速打印
